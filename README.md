# Language Learning Platform

## Introduction
The Language Learning Platform is a simple Java application that allows users to explore and start courses for learning different programming languages. It provides functionality to add courses, display available courses, and start a specific course.

## Features
- Add courses for different programming languages
- Display available courses
- Start a specific course

## How to Use
1. Clone or download the repository to your local machine.
2. Open the project in your preferred Java IDE.
3. Run the Main class to start the application.
4. Follow the on-screen instructions to add courses, display courses, and start a specific course.
